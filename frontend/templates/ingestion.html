pgAdmin Setup Guide
1
Install & Start pgAdmin
Download pgAdmin 4 from pgadmin.org or use Docker:
docker-compose up pgadmin
2
Configure .env Credentials
Edit .env with your PostgreSQL host, port, dbname, user, password.
3
Run Database Setup Script
Creates tables, indexes and bulk-loads both CSV files:
python setup_db.py
4
Verify in pgAdmin UI
Open pgAdmin → connect to DB → check tables shipments_master and route_events (5M rows each).
5
Start Live Stream
Go to Dashboard → click ▶ Start Live Stream. 5 real DB records stream every 10 seconds.
